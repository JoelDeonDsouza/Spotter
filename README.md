## Run Locally

Clone the project

```bash
  git clone https://github.com/JoelDeonDsouza/Spotter.git
```

Go to the project directory

```bash
  cd Spotter
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
